You're running Spaz for webOS, an open source microblogging client for Twitter, Idenit.ca, and StatusNet hosts.

If you find bugs or have ideas for features, let us know!

Spaz is created by volunteers. Any donations are used to cover hosting costs and compensate our contributors when possible.



Spaz is open source software with a New BSD-style license. We need volunteers to participate in the development process. If you're interested in helping, please join the Spaz Mobile Google Group.
